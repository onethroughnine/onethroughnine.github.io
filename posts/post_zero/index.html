Headings

Markdown supports multiple levels of headings using the hash symbol (#). This is a level 2 heading, and you can create up to level 6 headings by adding more hash symbols.

Text Formatting

You can make text bold by wrapping it with double asterisks or double underscores. You can make text italic by wrapping it with single asterisks or underscores. You can also combine them for bold italic text.

You can also use strikethrough for deleted text.

Lists

Unordered Lists

Unordered lists can be created using hyphens, asterisks, or plus signs:

  • First item
  • Second item
  • Third item
    • Nested item
    • Another nested item

Ordered Lists

Ordered lists use numbers followed by periods:

  1. First item
  2. Second item
  3. Third item
    1. Nested numbered item
    2. Another nested item

Links and Images

You can create inline links using square brackets for the text and parentheses for the URL.

You can also create reference-style links like this example, which references a link defined elsewhere.

Code

Inline code can be written using backticks: const x = 42;

For code blocks, use triple backticks with an optional language identifier:

function hello() {
  console.log("Hello, World!");
  return true;
}

def greet():
    print("Hello, World!")
    return True

Sometimes you need to display special characters literally. Use a backslash to escape them: *not italic* or [not a link].
